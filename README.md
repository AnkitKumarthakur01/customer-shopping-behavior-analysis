# 📊 Customer Shopping Behavior Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior using python, SQL, and Power BI data analytics techniques. The goal is to extract meaningful insights from customer data, understand purchasing patterns, and support better business decision-making.

The analysis includes revenue trends, customer segmentation, product performance, and the impact of discounts and subscriptions on customer spending.

---

## 🚀 Key Objectives

* Analyze customer purchasing behavior
* Identify revenue patterns across different demographics
* Understand the impact of discounts and subscriptions
* Segment customers based on buying history
* Discover top-performing products and categories

---

## 🛠️ Tech Stack

* **Database:** PostgreSQL
* **Language:** SQL
* **Tools:** Power BI / Tableau (for visualization)
* **Dataset:** Customer shopping behavior dataset

## 📜 Sample SQL Queries

```sql id="sql1"
-- Total revenue by gender
SELECT gender, SUM(purchase_amount) AS revenue
FROM customer
GROUP BY gender;
```

```sql id="sql2"
-- Top 5 highest rated products
SELECT item_purchased, ROUND(AVG(review_rating::numeric),2)
FROM customer
GROUP BY item_purchased
ORDER BY AVG(review_rating) DESC
LIMIT 5;
```
